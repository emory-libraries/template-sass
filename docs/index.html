    screens

    Since 1.0.0
    $screens: (
  
  's': max-width ($screen-s - 1),
  'm': $screen-s ($screen-l - 1),
  'l': min-width $screen-l
  
) !default;

    Description

    Compiles our commonly used breakpoints into a range that can used with Breakpoint to easily build @media queries. This map uses a schema compatible with Brandy's screen() function.

    Type

    Map

    Map structure

    map key Namemap key Descriptionmap key Typemap key Value
    s

    The main breakpoint for small screens

    		Mixedmax-width ($screen-s - 1)
    m

    The main breakpoint for medium screens

    		Mixed$screen-s ($screen-l - 1)
    l

    The main breakpoint for large screens

    		Mixedmin-width $screen-l

    Requires

    • [variable] screen-s
    • [variable] screen-m
    • [variable] screen-l
